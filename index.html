How it works

Three components. One pipeline.

Each component runs sequentially — profile in, prioritised lead out.

Component 01
Lead scorer
Random Forest + Gradient Boosting ensemble. 12 LinkedIn signals → Hot / Warm / Cold with 0–100 priority score and confidence %.
scikit-learn
Component 02
Reply predictor
TF-IDF + GBM regression. Scores message quality, predicts reply probability %, and gives specific tips to improve before sending.
NLP + GBM
Component 03
Agentic AI
Agent thinks → LLM writes message → autonomously decides SEND / REVIEW / SKIP. From 1000 leads to the right 60 on autopilot. Exactly like Ayush described.
LLM + Agent
